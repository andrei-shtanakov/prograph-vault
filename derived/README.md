# derived/ — пишут ИНСТРУМЕНТЫ

> ⚠️ **Руками не редактировать.** Содержимое перезаписывается инструментами. Ручные правки
> потеряются. Хочешь зафиксировать знание — оно идёт в `../authored/`.

## Подпапки

| Папка | Кто пишет | Что |
|---|---|---|
| `graph/` | prograph | структура проектов: deps, контракты, MCP-вызовы |
| `contracts/` | prograph / вендор-скрипты | **снапшоты** контрактов для поиска (НЕ авторитет — он в репе-производителе) |
| `projects/` | prograph | авто-факты по каждой репе |
| `digests/` | дайджест-скилы | claude-kb дайджесты |

## Правила

- Инструмент пишет **только** в свою подпапку `derived/`; запись в `../authored/` запрещена.
- Снапшот контракта помечается источником и хешем/версией оригинала (traceability до владельца).
- `derived/` ре-генерируется — не хранить здесь ничего, что нельзя воспроизвести из источников.
